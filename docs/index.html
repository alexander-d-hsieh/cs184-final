Final Project: Shattering Brittle Objects

Nicole Kinser (25271578), Alexander Hsieh (25452963), Derek Kennedy (25276436)

This will simulate a brittle object shattering against the ground. We will attempt to realistically simulate objects, such as glass and ceramic, falling to the ground and the ways in which they break.

Problem Description

This problem is important because it is hard for animators do draw frame by frame shattering, and it is a common enough action in animation that it should be easily replicated and produced, regardless of initial conditions.

Goals and Deliverables

What we plan to deliver

A generic brittle object shattering

What we hope to deliver

Objects of different materials shattering Objects shattering when being hit by a hard object (i.e. a ball, a hammer, etc)
