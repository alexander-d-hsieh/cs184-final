Project Summary

In this project we will simulate a brittle object shattering against the ground. We will attempt to realistically simulate objects such as glass and ceramic, falling to the ground and the ways in which they break.

Problem Description

This problem is important because it is hard for animators do draw frame by frame shattering, and it is a common enough action in animation that it should be easily replicated and produced, regardless of initial conditions.

Goals and Deliverables

What we plan to deliver

A generic brittle object shattering

What we hope to deliver

Objects of different materials shattering
