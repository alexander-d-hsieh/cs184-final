Constraints

The set of constraints between pointmasses can be represented by the vector of constraint functions \(C\) of the form \(C_i(p_a, p_b) = \left\lVert p_a - p_b \right\rVert - d_i\), which nonzero only if the constraint is broken i.e. the distance between the pointmasses \(p_a, p_b\) are no longer at distance \(d_i\). However, instead of using pointmass displacement as criteria for breaking the constraint, we give the constraint a maximum strength and calculate the force the constraint must exert when impace occurs, and if it is greater than the strength, we consider it broken.

In the fracturing simulation, we are estimating if the constraint is broken by finding the reaction forces applied by the pointmasses in response to the impact forces generated when the object hits the ground. We do this by solving this optimization problem

$$JWJ^T \lambda = -JWQ$$
where \(J = \frac{\partial C }{\partial p}\)

Here \(Q\) represents the impact force vector, \(W\) is the square matrix with elementwise inverse of each pointmass on the diagonals. And \(\lambda\) is our unknown vector.

Upon solving for Î», we can find the reaction force vector by finding \(\hat{Q} = J^T\lambda\). Since this is a vector of forces acted on the pointmasses, we project the forces to the vector \(p_a - p_b\) to find the magnitude of the force that the constraint is exerting in the direction between the two pointmasses.
